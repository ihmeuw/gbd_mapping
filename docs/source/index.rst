gbd_mapping has moved
=====================

This documentation site is **archived**. The ``gbd_mapping`` package has been
renamed and migrated into the
`vivarium-suite monorepo <https://github.com/ihmeuw/vivarium-suite>`_.

Which docs do you want?
-----------------------

If you have a specific version pinned (e.g. ``gbd_mapping==5.0.4``):

  Stay on this site and use the **version selector** to switch to your installed
  version. Those releases still ship the original ``gbd_mapping`` module and the
  per-version API reference is preserved for them.

If you are not pinned or you are otherwise on ``gbd_mapping >= 5.1.0``:

  You are running the **shim release**, which installs
  `vivarium-gbd-mapping <https://pypi.org/project/vivarium-gbd-mapping/>`_
  under the hood. The real code lives at the new import paths
  ``vivarium.gbd_mapping`` and ``vivarium.gbd_mapping_generator``.
  Go to https://vivarium-gbd-mapping.readthedocs.io/ for current docs.
